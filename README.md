# My Shakespeare Flashcards

A web-based flashcard application for learning and practicing Shakespearean figures of speech. This project is designed to help users familiarize themselves with the literary devices and rhetorical figures found in Shakespeare's works.

**Live Demo:** [bitdozer.github.io/my-shakespeare-flashcards](https://bitdozer.github.io/my-shakespeare-flashcards/)

## About

This site provides interactive flashcards to help users learn and identify figures of speech as used by Shakespeare. The figures of speech are based on the comprehensive list provided by [myshakespeare.me](https://myshakespeare.me/shakespeares-works/elements/figures-of-speech/). For more information about Shakespeare's works and educational resources, please visit [myshakespeare.me](https://myshakespeare.me/).
